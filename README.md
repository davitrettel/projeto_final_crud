# Sistema de Cadastro de Clientes

Este projeto é um sistema CRUD simples desenvolvido para cadastrar, listar, editar e excluir clientes.  
A aplicação foi criada com foco em praticar conceitos de desenvolvimento web, manipulação de dados e organização de interface.

## 📌 Sobre o projeto

O sistema permite gerenciar informações de clientes de forma simples e prática.  
Ele possui uma interface visual com tabela, botões de ação e campos de formulário para facilitar o cadastro e a alteração dos dados.

O projeto foi desenvolvido como atividade escolar/técnica para treinar os conceitos de CRUD:

- Create: cadastrar novos clientes;
- Read: visualizar os clientes cadastrados;
- Update: editar informações já salvas;
- Delete: excluir clientes da lista.

## 🚀 Funcionalidades

- Cadastro de clientes;
- Listagem dos clientes em uma tabela;
- Edição de dados cadastrados;
- Exclusão de registros;
- Interface organizada e responsiva;
- Estilização personalizada com CSS;
- Uso de Bootstrap para melhorar o visual da página.

## 🛠️ Tecnologias utilizadas

- HTML5;
- CSS3;
- JavaScript;
- Bootstrap;
- Git e GitHub.
